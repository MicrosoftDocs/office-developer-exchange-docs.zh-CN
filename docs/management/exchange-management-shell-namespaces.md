# <a name="exchange-management-shell-namespaces"></a>Exchange 命令行管理程序命名空间

在 Exchange 中找到有关使用的命名空间的 Exchange 命令行管理程序的信息。
  
**适用于：** Exchange Online |Exchange Server 2013 |Office 365
  
Exchange Online、 Exchange Online 作为 Office 365 的一部分或的开头 Exchange 2013 的 Exchange 版本的 Exchange 命令行管理程序应用程序交互通过由**公开的类型的 Windows PowerShell 环境System.Management.Automation**命名空间。 Exchange 命令行管理程序 cmdlet 从多个 Exchange 库使用类型。 本节提供有关的类型和使用 Exchange 命令行管理程序 cmdlet Exchange 库中的接口的参考信息。
