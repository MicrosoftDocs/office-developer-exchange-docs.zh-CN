查找 exchange 中 Exchange 命令行管理程序的命名空间的相关信息。
  
**适用于：** Exchange Online |Exchange Server 2013 |Office 365
  
Exchange Online 命令行管理程序 for Exchange Online、Exchange Online 作为 Office 365 的一部分，或者从 Exchange 2013 开始的 Exchange 版本通过由**System. Management** namespace 公开的类型与 Windows PowerShell 环境进行交互。 Exchange 命令行管理程序 cmdlet 使用大量的 Exchange 库中的类型。 本节提供了有关 exchange 库中的类型和接口的参考信息，这些信息由 Exchange 命令行管理程序 cmdlet 使用。
