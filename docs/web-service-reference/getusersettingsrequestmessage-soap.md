# <a name="getusersettingsrequestmessage-soap"></a>GetUserSettingsRequestMessage (SOAP)

**GetUserSettingsRequestMessage**元素均表示一个[GetUserSettings 操作 (SOAP)](getusersettings-operation-soap.md)请求。 
  
```XML
<GetUserSettingsRequestMessage>
   <Request/>
</GetUserSettingsRequestMessage>
```

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

|**元素**|**说明**|
|:-----|:-----|
|[请求 (SOAP)](request-soap.md) <br/> |包含请求的配置设置。  <br/> |
   
### <a name="parent-elements"></a>父元素

无。
  
## <a name="text-value"></a>文本值

无。
