# <a name="authenticationmethods-soap"></a>AuthenticationMethods (SOAP)

**AuthenticationMethods**元素描述了适用于 Web 客户端的身份验证方法。 
  
```XML
<AuthenticationMethods/>
```

 **string**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[WebClientUrl (SOAP)](webclienturl-soap.md) <br/> |代表 Outlook Web App 客户端的 URL。  <br/> |
   
## <a name="text-value"></a>文本值

**AuthenticationMethods**元素的文本值是在 Outlook Web App 客户端的 URL。
