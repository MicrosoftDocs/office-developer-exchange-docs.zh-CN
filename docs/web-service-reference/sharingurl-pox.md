# <a name="sharingurl-pox"></a>SharingUrl (POX)

**SharingUrl**元素包含用于跨组织共享日历和联系人信息的共享服务器的 URL。 
  
[自动发现 (POX)](autodiscover-pox.md)
  
[响应 (POX)](response-pox.md)
  
[帐户 (POX)](account-pox.md)
  
[协议 (POX)](protocol-pox.md)
  
[SharingUrl (POX)](sharingurl-pox.md)
  
```XML
<SharingUrl/>
```

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[协议 (POX)](protocol-pox.md) <br/> |包含客户端连接到运行 Microsoft Exchange Server 已安装了客户端访问服务器角色的计算机的规格。  <br/> |
   
## <a name="text-value"></a>文本值

文本值表示为跨组织共享日历和联系人使用的共享服务器的 URL。
  
## <a name="remarks"></a>备注

**SharingUrl**元素是**协议**元素的可选子元素。
