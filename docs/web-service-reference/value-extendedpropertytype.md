# <a name="value-extendedpropertytype"></a>值 (ExtendedPropertyType)

**Value**元素指定的角色扩展属性的数组。 
  
```XML
<Value>
   <ExtendedFieldURI/>
   <Value/>
   <Values/>
</Value>
```

**ExtendedPropertyType**

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

[ExtendedFieldURI](extendedfielduri.md) | [值](value.md) | [值](values.md)
  
### <a name="parent-elements"></a>父元素

[ExtendedPropertyAttributedValue](extendedpropertyattributedvalue.md)
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
