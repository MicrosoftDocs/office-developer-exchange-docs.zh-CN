# <a name="displayname-soap"></a>DisplayName (SOAP)

**DisplayName**元素均表示一个显示名称。 
  
```XML
<DisplayName/>
```

 **string**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[AlternateMailbox (SOAP)](alternatemailbox-soap.md) <br/> |代表一个备用的邮箱。  <br/> |
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |代表共享位置为文档的位置和元数据信息。  <br/> |
   
## <a name="text-value"></a>文本值

**DisplayName**元素的文本值是设置的显示名称。
