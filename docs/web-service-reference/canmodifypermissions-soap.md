# <a name="canmodifypermissions-soap"></a>CanModifyPermissions (SOAP)

**CanModifyPermissions**元素指示用户是否可以修改到文档共享位置的访问权限。 
  
```XML
<CanModifyPermissions /> 
```

 **Boolean**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[DocumentSharingLocation (SOAP)](documentsharinglocation-soap.md) <br/> |代表共享位置为文档的位置和元数据信息。  <br/> |
   
## <a name="text-value"></a>文本值

**CanModifyPermissions**元素的布尔值指示用户是否可以修改到共享位置的访问权限。 
  
## <a name="element-information"></a>元素信息

|||
|:-----|:-----|
|命名空间  <br/> |http://schemas.microsoft.com/exchange/2010/Autodiscover  <br/> |
|架构名称  <br/> |自动发现架构  <br/> |
|验证文件  <br/> |Messages.xsd  <br/> |
|可以为空  <br/> |True  <br/> |
