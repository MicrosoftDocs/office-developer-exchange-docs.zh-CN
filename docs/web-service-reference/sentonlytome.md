# <a name="sentonlytome"></a>SentOnlyToMe

**SentOnlyToMe**元素指示是否邮箱所有者必须是唯一的条件或例外应用的顺序中的传入消息的**ToRecipients**属性中。 
  
```XML
<SentOnlyToMe/>true | false</SentOnlyToMe>
```

 **Boolean**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[条件](conditions.md) <br/> |表示履行时将触发规则的规则操作的条件。  <br/> |
|[异常](exceptions.md) <br/> |代表收件箱规则的所有可用规则例外条件。  <br/> |
   
## <a name="text-value"></a>文本值

文本值为**true**指示邮箱的所有者必须是中的条件或例外应用的顺序的传入消息的**ToRecipients**属性中只能有一个。 如果值为**false**指示邮箱所有者不得中的条件或例外应用的顺序的传入消息的**ToRecipients**属性中只能有一个。 
  
## <a name="remarks"></a>备注

描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
