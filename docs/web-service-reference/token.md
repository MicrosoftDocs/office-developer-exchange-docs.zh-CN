本文档可能包含与预发布功能或产品相关的内容，这些功能或产品在最终商业发布之前可能会有重大变化。本文档按"原样"提供，仅供参考，Microsoft 不在本文档中作出任何明示或暗示担保。 **Token** 元素包含表示共享数据标识令牌的加密数据。 
  
```xml
<Token/>
```

 **EncryptedDataContainerType**

无。

|**元素**|**说明**|
