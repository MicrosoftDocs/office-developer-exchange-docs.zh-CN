# <a name="arrayofuserresponse-soap"></a>ArrayOfUserResponse (SOAP)

**ArrayOfUserResponse**元素包含[用户回音 (SOAP)](userresponse-soap.md)元素的数组。 
  
```XML
<ArrayOfUserResponse>
   <UserResponse/>
</ArrayOfUserResponse>
```

 **ArrayOfUserResponse**
