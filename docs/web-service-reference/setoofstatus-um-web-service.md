# <a name="setoofstatus-um-web-service"></a>SetOofStatus （UM web 服务）

**SetOofStatus**元素定义将统一消息外出 (OOF) 状态设置为发出请求的用户的请求。 
  
[SetOofStatus （UM web 服务）](setoofstatus-um-web-service.md)
  
```xml
<SetOofStatus>
  <status/>
</SetOofStatus>
```

 **类型**
