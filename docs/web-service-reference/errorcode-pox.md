# <a name="errorcode-pox"></a>ErrorCode （POX）

**ErrorCode**元素包含错误发现响应的错误代码。 
  
[错误（POX）](error-pox.md)


### <a name="attributes"></a>Attributes

无。
  
### <a name="child-elements"></a>子元素

无。

|**元素**|**说明**|
|:-----|:-----|
|[错误（POX）](error-pox.md) <br/> |包含自动发现错误响应。  <br/> |
