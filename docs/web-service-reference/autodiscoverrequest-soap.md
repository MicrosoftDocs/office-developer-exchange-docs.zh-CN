# <a name="autodiscoverrequest-soap"></a>AutodiscoverRequest (SOAP)

**AutodiscoverRequest**元素表示所有自动发现请求的基本的元素。 
  
```XML
<AutodiscoverRequest/>
```

### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

无。
  
## <a name="text-value"></a>文本值
