# <a name="additionalinfo"></a>AdditionalInfo

**AdditionalInfo**元素指定邮箱的其他信息的保留状态。 
  
```XML
<AdditionalInfo></AdditionalInfo>
```

 **将**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[MailboxHoldStatus](mailboxholdstatus.md) <br/> |指定邮箱的保留状态。  <br/> |
|[NonIndexableItemDetail](nonindexableitemdetail.md) <br/> |指定不能编制索引的项的详细信息。  <br/> |
   
## <a name="text-value"></a>文本值

AdditionalInfo 元素的文本值是有关的保留状态的邮箱的其他信息。
  
## <a name="remarks"></a>备注

此元素为可选。
  
Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
