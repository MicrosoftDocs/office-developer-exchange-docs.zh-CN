# <a name="sipresponsecode"></a>SIPResponseCode

**SIPResponseCode**元素指定的 SIP 响应代码。 
  
```xml
<SIPResponseCode/>
```

 **int**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[PhoneCallInformation](phonecallinformation.md) <br/> |指定的电话的状态信息。  <br/> |
   
## <a name="text-value"></a>文本值

SIP 响应代码。
  
## <a name="remarks"></a>注解

描述此元素的架构位于运行 Microsoft Exchange Server 2010 的安装了客户端访问服务器角色的计算机的 /ews/ 目录中。
