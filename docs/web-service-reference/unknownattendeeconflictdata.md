## <a name="remarks"></a>备注

Attendee 是未知的如果它无法解析对 Active Directory 目录服务对象。 如果无法确定为用户、 组或联系人与会者无法解析。 例如，与会者不会解析是否已启用邮件的公用文件夹。
  
描述此元素的架构位于正在运行 MicrosoftExchange Server 2007 的计算机（已安装客户端访问服务器角色）的 EWS 虚拟目录中。
