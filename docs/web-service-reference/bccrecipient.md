# <a name="bccrecipient"></a>BccRecipient

**BccRecipient**元素均表示接收的电子邮件的密件抄送 (Bcc) 收件人。 
  
```XML
<BccRecipient>true | false</BccRecipient>
```

 **Boolean**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[RecipientTrackingEvent](recipienttrackingevent.md) <br/> |包含收件人为一个事件的信息。  <br/> |
   
## <a name="text-value"></a>文本值

此元素可以是**true**或**false**。 值为**true**指示收件人是密件抄送复制;如果值为**false**指示收件人不是复制的密件抄送。 
  
## <a name="remarks"></a>备注

描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
