# <a name="generation"></a>生成

**生成**元素均表示的代缩写，按照联系人的完整名称。 
  
```xml
<Generation/>
```

 **字符串**

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
