# <a name="messageclassifications"></a>MessageClassifications

**MessageClassifications**元素均表示必须在应用的条件或例外顺序中的传入邮件标记邮件分类。 
  
```XML
<MessageClassifications>
    <String/>
</MessageClassifications>
```

 **ArrayOfStringsType**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无
  
### <a name="child-elements"></a>子元素

|**元素**|**说明**|
|:-----|:-----|
|[字符串](string.md) <br/> |代表邮件分类。  <br/> |
   
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[条件](conditions.md) <br/> |表示履行时将触发规则的规则操作的条件。  <br/> |
|[异常](exceptions.md) <br/> |表示表示收件箱规则的所有可用的规则例外条件的异常。  <br/> |
   
## <a name="text-value"></a>文本值

无。
  
## <a name="remarks"></a>备注

描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
