# <a name="postofficebox"></a>PostOfficeBox

**PostOfficeBox**元素指定邮政地址的"post office box"的部分。 
  
```XML
<PostOfficeBox></PostOfficeBox>
```

 **string**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

[PostalAddress (PersonaPostalAddressType)](postaladdress-personapostaladdresstype.md) | [值 (PersonaPostalAddressType)](value-personapostaladdresstype.md)
  
## <a name="text-value"></a>文本值

**PostOfficeBox**元素的文本值是邮政信箱标识符。 
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
