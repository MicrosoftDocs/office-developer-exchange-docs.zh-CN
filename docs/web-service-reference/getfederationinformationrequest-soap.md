# <a name="getfederationinformationrequest-soap"></a>GetFederationInformationRequest (SOAP)

**GetFederationInformationRequest**元素均表示调用[GetFederationInformation 操作 (SOAP)](getfederationinformation-operation-soap.md)操作的参数。
  
```XML
<GetFederationInformationRequest>
   <Domain/>
</GetFederationInformationRequest>
```

**GetFederationInformationRequest**

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。

|**元素**|**说明**|

|[域 (GetFederationInformation) (SOAP)](domain-getfederationinformationsoap.md) <br/> |标识了联合身份验证信任的域。  <br/> |

无。
