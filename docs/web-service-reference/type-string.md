# <a name="type-string"></a>类型 （字符串）

**Type**元素指定邮政地址或电话号码，例如，"主页"或"公司"的类型。 
  
```XML
<Type></Type>
```

 **string**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

[电话](phone.md) | [PhoneNumber](phonenumber.md) | [值 (PersonaPostalAddressType)](value-personapostaladdresstype.md) | [PostalAddress (PersonaPostalAddressType)](postaladdress-personapostaladdresstype.md)
  
## <a name="text-value"></a>文本值

**Type**元素的文本值是邮寄地址或电话号码的类型。 例如，"主页"或"公司"的值是预期的**Type**元素的值。 
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
