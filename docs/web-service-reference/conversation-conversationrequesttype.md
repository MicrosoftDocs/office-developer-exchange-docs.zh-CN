# <a name="conversation-conversationrequesttype"></a>对话 (ConversationRequestType)

**Conversation**元素表示**GetConversationItems**响应中返回单个对话。 
  
```XML
<Conversation>
   <ConversationId/>
   <SyncState/>
</Conversation>
```

 ****
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

[ConversationId](conversationid.md) | [SyncState (base64Binary)](syncstate-base64binary.md)
  
### <a name="parent-elements"></a>父元素

[Conversations](conversations-ex15websvcsotherref.md)
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
