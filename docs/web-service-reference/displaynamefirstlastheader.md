# <a name="displaynamefirstlastheader"></a>DisplayNameFirstLastHeader

**DisplayNameFirstLastHeader**元素指定显示名称的头（首先为名字）。 
  
```XML
<DisplayNameFirstLastHeader></DisplayNameFirstLastHeader>
```

 **string**
## <a name="attributes-and-elements"></a>属性和元素

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[角色](persona.md) <br/> |指定由**GetPersona**请求返回的一组角色数据。  <br/> |
   
## <a name="text-value"></a>文本值

**DisplayNameFirstLastHeader**元素的文本值是一个指定页眉的字符串值。 
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
