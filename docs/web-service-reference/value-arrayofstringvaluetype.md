# <a name="value-arrayofstringvaluetype"></a>值 (ArrayOfStringValueType)

**Value**元素数组的数组归属相关联的角色属性中指定的值。 
  
```XML
<Value></Value>
```

**string**

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

[值 (ArrayOfStringValueType)](values-arrayofstringvaluetype.md)
  
## <a name="text-value"></a>文本值

**Value**元素的文本值的数组归属相关联的角色属性数组中的值。 
  
## <a name="remarks"></a>备注

Exchange Server 2013 中引入了此元素。
  
描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
