# <a name="alternatemailboxcollectionsetting-soap"></a>AlternateMailboxCollectionSetting （SOAP）

**AlternateMailboxCollectionSetting**元素包含备用邮箱设置的集合。 
  
```XML
<AlternateMailboxCollectionSetting>
   <AlternateMailboxes/>
</AlternateMailboxCollectionSetting>
```

 **AlternateMailboxCollectionSetting**

下面各部分介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>Attributes
