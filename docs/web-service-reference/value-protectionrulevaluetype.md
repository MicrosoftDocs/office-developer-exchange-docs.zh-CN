# <a name="value-protectionrulevaluetype"></a>值 (ProtectionRuleValueType)

**Value**元素标识单个收件人或发件人部门。 
  
```XML
<Value/>
```

**ProtectionRuleValueType**

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[RecipientIs](recipientis.md) <br/> |指定任何收件人的电子邮件匹配的任何子**值**元素中指定的收件人。  <br/> |
|[SenderDepartments](senderdepartments.md) <br/> |指定发件人的部门匹配任何子**值**元素中指定的部门。  <br/> |
   
## <a name="text-value"></a>文本值

此元素必须包含一个非空字符串值。
  
## <a name="remarks"></a>备注

描述此元素的架构位于承载 Exchange Web Services 的 IIS 虚拟目录中。
