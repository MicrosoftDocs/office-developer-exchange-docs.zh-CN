# <a name="protocolconnections-soap"></a>ProtocolConnections （SOAP）

**ProtocolConnections**元素包含零个或多个协议连接。 
  
```XML
<ProtocolConnections>
   <ProtocolConnection/>
</ProtocolConnections>
```

 **ProtocolConnections**
