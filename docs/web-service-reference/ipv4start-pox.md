# <a name="ipv4start-pox"></a>IPv4Start (POX)

**IPv4Start**元素标识的范围内的 IP 版本 4 (IPv4) 开始用于标识网络上的计算机的地址。

[IPv4Start (POX)](ipv4start-pox.md)
  
```xml
<IPv4Start/>
```

## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[NetworkRequirements (POX)](networkrequirements-pox.md) <br/> |包含用于确定客户端计算机是否满足 Internet 服务提供商 (ISP) 的网络上的标准要求连接到服务器。  <br/> |
   
## <a name="text-value"></a>文本值

文本值表示 IPv4 地址。
