# <a name="status-um-web-service---setoofstatus"></a>状态 （UM web 服务-SetOofStatus）

**Status**元素定义在[SetOofStatus 操作 （UM web 服务）](setoofstatus-operation-um-web-service.md)请求中使用的值。 
  
[SetOofStatus （UM web 服务）](setoofstatus-um-web-service.md)
  
[状态 （UM web 服务-SetOofStatus）](status-um-web-servicesetoofstatus.md)
  
```xml
<SetOofStatus>
  <status/>
</SetOofStatus>
```

 **Boolean**
## <a name="attributes-and-elements"></a>属性和元素

如下章节中介绍了属性、子元素和父元素。
  
### <a name="attributes"></a>属性

无。
  
### <a name="child-elements"></a>子元素

无。
  
### <a name="parent-elements"></a>父元素

|**元素**|**说明**|
|:-----|:-----|
|[SetOofStatus （UM web 服务）](setoofstatus-um-web-service.md) <br/> |定义将统一消息外出 (OOF) 状态设置为发出请求的用户的请求。  <br/> |
   
## <a name="text-value"></a>文本值

需要安装一个布尔值。 以下是可能的值：
  
- True
    
- False
